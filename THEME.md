# 水浒典藏主题

古典中国风主题，专为水浒传人物卡片设计的宣纸水墨风格。

## Color Palette

- **古籍褐**: `#8B4513` - 主色调，赭石色，用于边框、标题
- **鎏金**: `#DAA520` - 强调色，用于星级、印章、书名
- **宣纸米**: `#F5F5DC` - 背景色，营造古籍质感
- **墨色**: `#2F4F4F` - 深色文字，稳重典雅
- **朱砂**: `#8B0000` - 点缀色，用于印章、座次、警示
- **浅米**: `#E8DCC0` - 卡片背景渐变

## Design Elements

### 卡片设计
- 3:4 竖版比例，仿古铜边框
- 宣纸纹理背景 + 水墨晕染效果
- 五星金色浮雕评级
- 竖排文字区（从右至左）
- 朱红印章座次标识
- 3D Y轴翻转动画

### 页面设计
- 渐变宣纸背景
- 胶囊形筛选按钮
- 卡片网格布局 + 入场动画
- 云纹装饰元素
