# 水浒知识问答集卡 🏯

> 一款融合知识问答与卡牌收集的古典文学互动体验游戏，以水墨宣纸风格呈现《水浒传》108将的英雄风采。

## ✨ 项目特色

### 🎨 独特的水墨视觉风格
- **宣纸纹理背景**：模拟古籍质感，营造沉浸式阅读体验
- **书法字体呈现**：采用马善政毛笔楷体，彰显传统文化韵味
- **古典配色方案**：古籍褐、鎏金、朱砂等传统色彩
- **3D翻转卡片**：Y轴翻转动画，正面立绘、背面详情

### 🎮 核心玩法

#### 1. 答题解锁英雄
通过回答与水浒英雄相关的知识题目，逐步解锁108将卡片。题目涵盖：
- 人物绰号与排名
- 经典故事情节
- 武器与技能
- 人物关系与结局

#### 2. 多维度排序浏览
解锁后的英雄卡片支持三种排序方式：
- **按座次**：天罡地煞排名
- **按武力**：武力值高低
- **按星级**：综合评级（1-5星）

#### 3. 无尽挑战模式
测试你的水浒知识储备：
- 连续答题挑战
- 答错即结束，挑战最高连胜纪录
- 实时排行榜，与好友比拼

#### 4. 成就系统（34个成就）
四大类成就等你解锁：

| 类别 | 数量 | 示例 |
|------|------|------|
| 🃏 卡片收集 | 11个 | 初上梁山、天罡归位、震古烁今 |
| 📝 答题闯关 | 10个 | 一气呵成、连中三元、百折不挠 |
| ⚔️ 无尽挑战 | 12个 | 横扫千军、天下无敌、闪电思维 |
| 🏆 特殊荣誉 | 8个 | 及时雨、黑旋风、鼓上蚤 |

每个成就配有铜/银/金三级徽章，达成后获得专属称号。

#### 5. 称号系统
- 根据成就解锁动态获得称号
- 可在成就殿堂查看当前称号
- 称号带有徽章等级标识（🥉🥈🥇）

#### 6. 图鉴浏览
- 天罡星/地煞星分类查看
- 已解锁/待解锁状态区分
- 英雄详情弹窗：生平事迹、结局、武器技能

## 📦 本地部署

```bash

git clone https://github.com/leon24kb/Shui-Hu-Trivia-Cards.git

# 进入目录
cd Shui-Hu-Trivia-Cards

# 启动本地服务器（任选一种）

# 或
npx serve .
# 或

# 浏览器访问
```

## 🛠️ 技术栈

- **前端**：原生 HTML5 + CSS3 + JavaScript（ES6+）
- **样式**：CSS 变量主题系统、Flexbox/Grid 布局
- **动画**：CSS3 Transitions & Animations
- **存储**：LocalStorage（进度本地保存）
- **字体**：Google Fonts（马善政毛笔楷体、思源宋体）
- **部署**：GitHub Pages + GitHub Actions
